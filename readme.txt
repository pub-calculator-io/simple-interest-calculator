The simple interest calculator can be used to provide a quick and accurate calculation of interest assessed on a loan or investment.

[https://www.calculator.io/simple-interest-calculator/](https://www.calculator.io/simple-interest-calculator/)

== Usage ==

Add the Simple Interest Calculator shortcode to your page, post or sidebar:

`[ci_simple_interest_calculator]`

Add the following code to your template where you would like the Simple Interest Calculator to appear:

`<?php display_ci_simple_interest_calculator(); ?>`

1. The Simple Interest Calculator Input Form.

== Installation ==

1. Upload the Simple Interest Calculator /ci_simple_interest_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Simple Interest Calculator plugin through the "Plugins" menu in WordPress.
